DeepSeek AI Chat connects your WordPress site to the DeepSeek AI API, allowing seamless AI-powered conversations with cross-page persistence.

== Description ==

DeepSeek AI Chat for WP Admin is a powerful WordPress plugin that integrates with the **DeepSeek AI API** to provide an interactive chatbot experience.  

**Key Features:**
- ?? **API Key Authentication** – Users must generate an API key from [DeepSeek.com](https://platform.deepseek.com/api_keys) and enter it in the settings.  
- ?? **Persistent Chat** – Conversations remain active across different pages.  
- ?? **Export Conversations** – Users can **download chat logs** as a text file.  
- ?? **Email Chat Logs** – Send chat history to an email of your choice.  
- ?? **More Features Coming Soon** – Stay tuned for updates with new functionalities!  

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to **Settings ? DeepSeek AI Chat** and enter your API key.
4. Save changes and start chatting!

== Frequently Asked Questions ==

= Where do I get an API key? =
You need to create an account at [DeepSeek.com](https://deepseek.com) and generate an API key from your dashboard.

= Does the chat persist across pages? =
Yes! The chat remains active as users navigate your website.

= Can I export chat logs? =
Absolutely! You can download chat history as a text file or send it to an email address.

= Will more features be added? =
Yes, we are actively working on **new enhancements and functionalities**. Stay updated!
